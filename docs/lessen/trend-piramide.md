# Trendpiramide

Trends bestaan op verschillende niveaus. Gebruik de piramide om trends van micro tot mega te ordenen:

```
        /\
       /  \
      / MEGA \        <-- Maatschappelijke megatrends
     /--------\
    /   MACRO   \     <-- Consumenten- en markttrends
   /--------------\
  /     MICRO      \  <-- Sector- en branchetrends
 /------------------\
/  OVERIGE DYNAMIEK  \  <-- Directe omgeving, concurrentie etc.
\--------------------/
```

| Niveau | Focus |
|--------|-------|
| **Mega** | Wat verandert er in de maatschappij? |
| **Macro** | Wat verandert er in consumentengedrag? |
| **Micro** | Wat gebeurt er in onze sector? |
| **Overig** | Directe omgeving, concurrentie, etc. |
