# Problem Clarification Template (AT1)

Vergelijk je verbeteropgave aan de hand van de vragen met een situatie waarin dit probleem niet voorkomt of in mindere mate. Vergelijk daarbij steeds de verschillen tussen het probleem en het non-probleem. *(Andler, 2016)*

---

| Vraag | Probleem | Non-probleem | Discrepantie |
|-------|----------|-------------|-------------|
| **Wie** heeft het probleem? Bij wie wordt het effect zichtbaar? | | Bij welke vergelijkbare groep treedt het effect niet op? | Wat zijn de verschillen? Wat kan dit betekenen richting een oplossingsrichting? |
| **Waar** treedt dit probleem op? Wat zijn de kenmerken van de situatie? | | Waar treedt dit probleem niet op? Wat kenmerkt die situatie? | Waar zitten de verschillen? Waar zou een oplossing nog meer nuttig zijn? |
| **Wanneer** treedt dit probleem op? Welke momenten, fasen of periodes? | | In welke vergelijkbare periodes speelt het niet? | Wat was het verschil? Wanneer moet dit zijn opgelost? |
| **Wat** is het probleem? Waar zit de pijn/belemmering? | | Wat is NIET het probleem? Wat is geen onderdeel van de kern? | Wat is er al geprobeerd? Wat is er veranderd? Wat moet anders? Wat als we niks doen? |
| **Hoe groot** is het probleem? Ernst, urgentie, omvang? | | Hoeveel worden NIET geraakt? | Wat is de omvang, ernst en urgentie? Hoe ziet een oplossing eruit? |
