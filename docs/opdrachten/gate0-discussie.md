# Gate 0 - Discussieformulier: Understanding the Problem

---

## Doel

In Gate 0 gebruik je je ontdekkingsreis (understanding the problem) om samen met anderen te zoeken naar de grote golven die ruimte kunnen bieden voor innovatie.

### Proces:
| Stap | Actie |
|------|-------|
| **Divergeren** | Breed trendonderzoek |
| **Convergeren** | Selecteer 6-9 trends |
| **Convergeren** | Kies maximaal 2 (te combineren) trends |

---

## Discussievragen per trend

Gebruik de volgende vragen om de discussie te richten. Gebruik de score pas **na** de discussie. Het doel is om door de ander de challenges beter te begrijpen.
